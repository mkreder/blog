<p>La nueva versión Alpha de Fedora 13 ya esta disponible para <a href="http://fedoraproject.org/get-prerelease">dercargar</a>!</p>
<p>Algunos de sus nuevos Features son:</p>
<ul>
<li>Instalación automática del driver de cualquier impresora</li>
<li>Instalación automática de paquetes de idioma</li>
<li>Herramientas para el manejo de usuarios rediseñadas</li>
<li>Manejo de Color en GNOME </li>
<li>Mejoras en NetworkManager, entre ellas un CLI</li>
<li>3D experimental extendido en el driver Nouveau para placas Nvidia</li>
</ul>
<p>Para desarroladores:</p>
<ul>
<li>Pruebas estáticas para SystemTap</li>
<li>Debugging de Python fácil</li>
<li>Instalación paralela de Python 3</li>
<li>NetBean 6.8 el primer IDE que soporta Java EE 6.0</li>
</ul>
<p>Para administradores:</p>
<ul>
<li>boot.fedoraproject.org</li>
<li>System Secury Srevices Daemon</li>
<li>Más features para NFS v4</li>
<li>Zarafa Groupware </li>
<li>Integración de snpashots en Btrfs</li>
</ul>
<p>Pueden leer mas acerca de este anuncio y cada feature <a href="http://fedoraproject.org/wiki/Releases/13/FeatureList?anF13a">aquí. </a></p>
